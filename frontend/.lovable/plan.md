

# 🏥 医疗解析与疾病管理智能体

## 设计风格
- **苹果液态玻璃 (Liquid Glass)** 风格：半透明毛玻璃卡片、柔和阴影、圆角、微妙的光泽渐变
- **配色方案**：以柔和的蓝紫色系为主调，搭配白色/浅灰背景，营造专业、舒适、安心的医疗氛围
- 大量使用 `backdrop-blur`、渐变透明背景和细腻的边框光泽效果

---

## 页面 1：品牌首页

- **顶部导航栏**：毛玻璃效果，Logo + 产品名 + 「开始使用」按钮
- **Hero 区域**：大标题介绍产品功能（如"上传病历，AI 为您解读"），配合柔和的渐变背景和医疗相关插图图标
- **功能介绍卡片**（3个玻璃质感卡片）：
  - 📄 病历解析 — 上传病历文档，智能解读诊断与用药
  - 🩻 影像分析 — 上传影像照片，获取通俗易懂的解释
  - 📋 疾病管理 — 长期跟踪病情，获取个性化建议
- **底部行动号召**：引导用户进入聊天页面

## 页面 2：AI 聊天页

- **全屏聊天界面**，玻璃质感消息气泡
- **消息类型**：
  - 用户文本消息
  - 用户上传图片/文件（支持拖拽或点击上传病历、影像照片）
  - AI 回复消息（模拟的医学分析结果，使用 Markdown 渲染，包含结构化的分析报告样式）
- **输入区域**：底部输入栏，附带上传按钮（📎）和发送按钮
- **侧边栏/历史**：左侧可展开的会话历史列表（模拟数据）
- **AI 模拟响应**：上传图片后返回模拟的分析报告（如检查指标解读、诊断建议等），打字机效果逐字显示

## 用户认证（使用 Lovable Cloud）

- 登录/注册页面，支持邮箱注册
- 简洁的玻璃质感登录表单
- 登录后跳转至聊天页
- 未登录用户访问聊天页自动跳转到登录

## 数据存储（后续阶段）

- 用户 profiles 表存储头像和昵称
- 聊天会话和消息的历史记录表
- 首先用前端模拟数据展示完整体验，后续再接入真实存储
