# medgemma 医疗助手

## 目标
开发一个基于langgraph和llm实现的医疗助手，能够提供以下功能：
1. 症状分析：用户输入症状（文字+图片），助手分析可能的疾病并提供建议。
2. 药物推荐：根据用户的症状和病史，推荐适合的药物和治疗方案。
3. 医疗咨询：用户可以咨询医疗相关问题，助手提供专业的解答。
4. 健康管理：提供健康管理建议，如饮食、运动等。
## 技术栈
- 前端：React.js
- 后端：python + fastAPI
- 数据库：PostgreSQL
- LLM：Ollama/OpenAI API
- websearch：Tavily API
## 功能模块设计
### 1. supervisor
负责整体流程控制，协调各个模块的工作，确保系统的稳定性和效率。

### 2. planner
负责根据用户输入的症状和病史，制定分析和推荐的计划，调用相应的工具和模型进行处理。
